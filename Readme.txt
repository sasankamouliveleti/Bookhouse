For running the application follow these :

1) open phpmyadmin in browser and create a database "book".

2) Place the Lays folder in www folder of wamp (htdocs for xampp) 
3) In browser URL type - localhost/BookHouse/create_table.php (This will create the required tables)
 

4) Now everyhting is set and you can run the solution by typing in URL localhost/Lays/pre.php
